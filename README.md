# r-ecommerce

Opinionated sample e-commerce platform built with **R**:
- Shiny front-end (app/)
- Plumber REST API (api/)
- SQLite database (db/)
- Docker + docker-compose for local dev
- Unit tests with `testthat`
- Simple mock payment gateway and analytics
